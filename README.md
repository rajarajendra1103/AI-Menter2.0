# 🚀 AI Menter 2.0: The Future of Interactive Coding Education

AI Menter 2.0 is an advanced, AI-powered learning platform designed to bridge the gap between abstract code and deep understanding. By combining high-speed AI orchestration with cutting-edge visualisations, we turn static code into dynamic, interactive experiences.

---

## 🌟 Key Features

### 🧠 AI-Powered Learning (AiLearning)
- **Guided Roadmaps**: Automatically generated learning paths for Beginner, Intermediate, and Advanced topics.
- **AI Tutoring**: In-depth explanations of coding concepts powered by **Llama 3.3 (Groq API)**.
- **Interactive Flashcards**: Reinforce learning with AI-generated review cards.

### 🎨 Visualisation Pipeline
- **Code Execution Flowcharts**: Real-time logic mapping using **Mermaid.js**.
- **Data Flow Trees**: Hierarchical tree graphs powered by **D3.js** that track state changes and value propagation.
- **Sequential Step Traces**: A precise breakdown of every line executed with its associated state.

### 🎬 Code Animations (2D/3D)
- **3D Logic Visualization**: A Three.js-powered arena that animates algorithms in a 3D space.
- **2D Dynamic Sketches**: Quick, data-driven animations for common data structures (Arrays, Lists, etc.).

### 💻 Advanced Code Editor
- **Multi-language Support**: Python, JavaScript, C++, and Java.
- **One-Click Visualisation**: Instantly generate graphs and animations for your current snippet.

---

## 🛠️ Technology Stack

- **Frontend**: React (Vite), Tailwind CSS
- **AI Orchestration**: Groq API (High-performance Llama 3.3 / 3.1 models)
- **Visualisation**: D3.js, Mermaid.js, Three.js
- **Icons & Styling**: Lucide React, Framer Motion

## 🚀 Getting Started

### Prerequisites
- [Node.js](https://nodejs.org/) (v18+)
- A [Groq API Key](https://console.groq.com/)

### Installation

1. **Clone the repository**:
   ```bash
   git clone https://github.com/rajarajendra1103/AI-Menter2.0.git
   cd AI-Menter2.0
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Configure Environment Variables**:
   Create a `.env` file in the root directory:
   ```env
   VITE_GROQ_API_KEY=your_groq_api_key_here
   VITE_SITE_NAME="AI Menter"
   ```

4. **Run the development server**:
   ```bash
   npm run dev
   ```
